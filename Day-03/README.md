# Day 3 - String Functions, ALTER TABLE & CASE Statement 📝

On Day 3, I learned how to manipulate text using SQL string functions, modify existing table structures with the `ALTER TABLE` statement, and apply conditional logic using the `CASE` statement in PostgreSQL.

---

## 📚 Topics Covered

### String Functions

- UPPER()
- LOWER()
- LENGTH()
- SUBSTRING()
- LEFT()
- RIGHT()
- CONCAT()
- CONCAT_WS()
- TRIM()
- REPLACE()

### ALTER TABLE

- ADD COLUMN
- DROP COLUMN
- RENAME COLUMN
- ALTER COLUMN TYPE
- SET DEFAULT
- ADD CONSTRAINT
- DROP CONSTRAINT
- RENAME TABLE

### CASE Statement

- CASE in SELECT
- CASE with UPDATE
- Conditional Data Categorization
- Practice Questions

## 🎯 Learning Outcomes

After completing Day 3, I can:

- Manipulate text using SQL string functions.
- Modify existing database tables.
- Add, rename, and remove columns.
- Change data types and constraints.
